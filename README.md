# my-smart-campus-app

Monorepo for Smart Campus web application.

Folders:
- `backend/` - Spring Boot application (Maven)
- `frontend/` - React (Vite) application

Quick start (backend):

1. Open `backend/` in your IDE (requires Java 17+ and Maven).
2. Build and run: `mvn spring-boot:run`

Quick start (frontend):

1. Open `frontend/` in a terminal (Node.js 18+ recommended).
2. Install deps: `npm install`
3. Run dev server: `npm run dev`
